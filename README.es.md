# Escriba de Código

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Code Ghostwriter para OpenClaw.

## Qué Resuelve
Escribe automáticamente documentación limpia y comentarios para bases de código existentes.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque de seguridad primero
- Soporte para reversión

## Uso
/code-ghostwriter

## Ejemplo
```
Analizar el código en busca de problemas de documentación.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
